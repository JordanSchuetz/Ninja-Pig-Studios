  Learn CS8 is an educational website used by the University of California Riverside to teach introductory computer programming to students not in an engineering major. Over 700 students each academic quarter use the Learn CS8 portal to learn HTML and CSS programming. Alongside programming tutorials, the website also features a comprehensive Google Drive walkthrough which teaches students how to use Microsoft Office alternatives which are located in the cloud.
